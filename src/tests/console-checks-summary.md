## Консольные команды, ставшие тестами

### Основные проверки циклических ссылок:
```bash
node -e "const obj = { name: 'John' }; obj.self = obj; console.log(obj.self === obj)"
```
→ **Тест:** `should handle simple self-reference (obj.self = obj)`

### Проверки формата сериализации:
```bash
node -e "const obj = { name: 'John' }; obj.self = obj; console.log(JSON.stringify(obj))"
```
→ **Тест:** `should produce correct $ref format for simple circular reference`

### Проверки Context:
```bash
node -e "const Context = require('./lib/context').Context; const ctx = new Context({name: 'test'}); console.log(ctx.toJSON())"
```
→ **Тест:** `should handle Context.toJSON() with circular references safely`

### Проверки полного цикла:
```bash
node -e "const {CyclicJSON} = require('./lib/JSON'); const obj = {name: 'test'}; obj.self = obj; const s = CyclicJSON.stringify(obj); const p = CyclicJSON.parse(s); console.log(p.self === p)"
```
→ **Тест:** `should maintain object identity through full cycle`
